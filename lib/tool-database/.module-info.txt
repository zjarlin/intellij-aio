模块名称: tool-database
模块类型: 工具类库模块
创建日期: 2025-11-23

【功能概述】
封装 IntelliJ IDEA Database 插件的工具类，提供便捷的数据库操作接口。

【主要类】
1. IntellijDataSourceManager - 数据源管理器
   - 获取用户在 Database 插件中配置的数据库连接
   - 支持按名称、ID、索引查找数据源

2. IntellijDataBaseSqlExecutor - SQL 执行器
   - 执行任意 SQL 语句
   - 支持查询、更新、批量操作
   - 异步执行，支持超时控制

3. SqlExecutionResult - 执行结果封装
   - 成功/失败状态
   - 查询结果数据
   - 错误信息
   - 执行时间统计

【使用前提】
- 用户需在 IntelliJ IDEA 中安装并启用 Database 插件
- 用户需在 Database 工具窗口中配置至少一个数据源

【技术特点】
- 异步执行，不阻塞 UI 线程
- 完善的错误处理和异常捕获
- 支持超时控制
- 线程安全
- 连接由 IntelliJ 管理，无需手动释放
