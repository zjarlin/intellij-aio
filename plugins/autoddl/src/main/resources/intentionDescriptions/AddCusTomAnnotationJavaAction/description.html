为代码添加自定义注解。 Add custom annotation to your code.

功能说明： Features:

  • 支持为代码添加自定义的注解 Support adding custom annotations to code
  • 可通过设置自定义注解模板 Can be configured through custom annotation templates

使用方法： How to use:

  1. 将光标放在需要添加注解的代码上 Place cursor on the code where you want to add annotation
  2. 按下 Alt+Enter Press Alt+Enter
  3. 选择 "Add Custom Annotation" Select "Add Custom Annotation"
  4. 自动添加自定义注解 Custom annotation will be added automatically

注意：请确保已在设置中配置了正确的自定义注解模板 Note: Please ensure custom annotation template is properly configured in settings
