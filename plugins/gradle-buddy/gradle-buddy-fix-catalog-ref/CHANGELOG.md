# Changelog

## [1.0.0] - 2026-01-23

### Added
- 初始版本发布
- 支持检测 Gradle 版本目录中的无效依赖引用
- 自动扫描 TOML 文件，提取依赖别名
- 在编辑器中用黄色波浪线标识错误引用
- 提供智能快速修复，自动替换为正确的引用
- 支持批量修复项目中所有相同的错误引用
- 使用编辑距离算法查找最佳匹配
- 支持多种版本目录名称（libs、zlibs、klibs 等）

### Features
- 自动转换命名规则：
  - 连字符和下划线转换为点分隔
  - 特殊处理 `gradle-plugin-` 前缀
- 智能匹配算法：
  - 精确匹配（忽略大小写）
  - 部分匹配
  - 编辑距离最小匹配
