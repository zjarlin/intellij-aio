Fetches the latest version from Maven Central and updates the version catalog dependency.

This intention is available when the caret is on a library declaration in a libs.versions.toml file, such as junit-jupiter-api = { group = "org.junit.jupiter", name = "junit-jupiter-api", version.ref = "jupiter" } or okhttp = { module = "com.squareup.okhttp3:okhttp", version = "4.12.0" }.

If the library uses version.ref, the version in the [versions] section will be updated. If no version reference exists, a new one will be generated and added automatically.
