# Dynamic K2 companion plugin that surfaces spread-pack overloads in JetBrains IDEs

Provides lightweight IDE-side visibility for overloads generated by `site.addzero.kcp.spreadpack`.

Current scope:

- builds dynamic K2 stubs for functions annotated with `@GenerateSpreadPackOverloads`
- mirrors current spread-pack flattening rules for `@SpreadPack` and `@SpreadArgsOf`
- supports top-level and member overload generation without bundling compiler plugin classes into the IDE process

Current non-goals:

- full compiler-diagnostic parity with the KCP backend
- rich navigation, quick fixes, or refactoring-aware synthetic declarations
- cross-process loading of the compiler plugin jar inside the IDE
