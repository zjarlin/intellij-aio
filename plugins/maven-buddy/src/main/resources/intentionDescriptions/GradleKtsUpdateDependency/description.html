Updates the Gradle KTS dependency version to the latest available version from Maven Central.

Supported formats:

  • Gradle Kotlin DSL: implementation("group:artifact:version")
  • Gradle Kotlin DSL: api("group:artifact:version")
  • Gradle Kotlin DSL: testImplementation("group:artifact:version")
  • And other Gradle dependency configurations

Place the cursor on a Gradle KTS dependency declaration in a .gradle.kts file and press Alt+Enter to see this intention.
