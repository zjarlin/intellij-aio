# Open Project Everywhere

Search and open repositories directly from Search Everywhere.

## Features

- Press `Shift` twice and search local projects under `~/IdeaProjects`.
- Search GitHub, GitLab, Gitee, and one custom Git host in parallel.
- Show the repository source category for every result.
- Open local matches directly or clone remote repositories into the configured local root and open them.
- Reuse credentials from this plugin for Git HTTPS clone operations.

## Configuration

Open `Settings | Tools | Open Project Everywhere` and configure any combination of:

- Local / clone root, defaulting to `~/IdeaProjects`
- GitHub
- GitLab
- Gitee
- Custom Git host

Token authentication is recommended. Username/password mode is available for self-hosted or compatible servers.
