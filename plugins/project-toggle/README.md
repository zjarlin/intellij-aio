# Display the list of open projects in the status bar at the bottom of the IDE, and click to switch.

在 IDE 底部状态栏显示打开的项目列表，点击切换。

## 效果

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                        编辑器区域                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│ ... │ [ProjA] [ProjB] [Current▁] [+] │ Git:main │ 行:1 列:1 │  ← 底部状态栏
└─────────────────────────────────────────────────────────────┘
              ↑ 项目名称显示在这里
```

## 功能

- **底部显示**: 项目按钮显示在状态栏（和 Git、行号位置相同）
- **点击切换**: 点击项目名称切换到对应项目窗口
- **当前项目标识**: 当前激活项目有蓝色下划线
- **+ 按钮**: 打开新项目
- **无悬浮按钮**: 不像 ToolWindow 那样有额外的标题栏按钮

## 使用方法

1. 安装插件后，查看 IDE 底部状态栏
2. 状态栏会显示 `| [项目1] [项目2] [+] |`
3. 点击项目名切换窗口

**如果没有显示：**
1. 检查 Settings → Tools → Project Tabs → Enable 是否勾选
2. 右键点击状态栏，确保 "Project Tabs" 已启用
3. 重启 IDE

## 设置

Settings → Tools → Project Tabs:
- **Enable**: 启用/禁用

## 技术说明

- 使用 StatusBarWidget 实现
- 显示在 IDE 底部状态栏
- 无额外悬浮控件
