根据字段注释中的枚举值定义自动生成对应的枚举类。 Generate enum class based on field comment's enum value definitions.

支持的注释格式： Supported comment formats:

  • 短横线格式：1-男, 2-女 Hyphen format: 1-Male, 2-Female
  • 冒号格式：1:男, 2:女 Colon format: 1:Male, 2:Female
  • 等号格式：1=男, 2=女 Equals format: 1=Male, 2=Female

使用方法： How to use:

  1. 在字段上添加包含枚举值定义的注释 Add comment with enum value definitions on the field
  2. 将光标放在字段上，按下 Alt+Enter Place cursor on the field and press Alt+Enter
  3. 选择 "GenEnumByFieldComment" Select "GenEnumByFieldComment"
  4. 自动生成对应的枚举类 Automatically generate corresponding enum class

示例： Example: 

// Java
/** 性别：1-男, 2-女 */
private Integer gender;

// Kotlin
/** 性别：1-男, 2-女 */
var gender: Int? = null

将生成： Will generate: 

enum class Gender(val code: Int, val description: String) {
    MALE(1, "男"),
    FEMALE(2, "女");

    companion object {
        fun of(code: Int): Gender = values().find { it.code == code }
            ?: throw IllegalArgumentException("Invalid Gender code: $code")
    }
}

注意：确保注释格式正确，且代码值和描述文本不重复 Note: Ensure comment format is correct and code values and descriptions are unique
